    機器學習基本概念簡介(2)

    學習時間 : 2026 / 03 / 13

    有時候，linear models 或許太簡單了，像遇到 Piecewise Linear Curve (一堆線段組成的) 時就不行了，這時，我們可以用常數 + 一堆函數 f。這個函數一開始是平的，然後變斜的，最後再變回平的。

    如果是曲型的怎麼辦?這時可以在曲型上點很多個點，連起來成為 Piecewise Linear Curve。這個 Piecewise Linear Curve 十分逼近這個曲線。

    至於第一段提到的函數 f，則是可以用一條曲線來逼近他。這個曲線是 :

    \[ y = c \times \frac{1}{1 + e^{-(b + wx_1)}} \]

    當 x 趨近無窮大時，y 會趨近於 c 當 x 趨近無窮小時，y 會趨近於 0

    此外，這個式子又可以寫為

    \[ y = c \times sigmoid(b + w \times x_1) \]

    然後這個函數 f 叫做 Sigmoid。

    所以說呢，Piecewise Linear Curve 可以表示成

    \[ y = b + \sum_i c_i \times sigmoid(b_i + w_i \times x_i) \]

    那如果有多個 feature 呢? 像是 \(y = b + \sum_j w_j x\)，這式子可以寫成

    \[ y = b + \sum_i c_i \times sigmoid(b_i + \sum_j w_{ij} \times x_j) \]

    j 就是 feature i 則是 函式 f

    然後上述式子可以變成矩陣的形式，最後所以未知的參數稱為 \(\theta\)。

    Loss

    現在定義 loss 為 \(L(\theta)\)。 但實際計算的方式跟上一篇一樣

    最佳化

    這裡有個矩陣 g 定義為 :

    \[ \text{gradient} \quad \boldsymbol{g} = \begin{bmatrix} \frac{\partial L}{\partial \theta_1} \bigg|_{\boldsymbol{\theta}=\boldsymbol{\theta}^0} \\[1em] \frac{\partial L}{\partial \theta_2} \bigg|_{\boldsymbol{\theta}=\boldsymbol{\theta}^0} \\[1em] \vdots \end{bmatrix} \]

    簡寫為

    \[ \boldsymbol{g} = \nabla L(\boldsymbol{\theta}^0) \]

    更新的方法跟上一篇的也一樣，只是變成一次很多個參數更新

    \[ \boldsymbol{\theta}^1 \leftarrow \boldsymbol{\theta}^0 - {\eta} \boldsymbol{g} \]

    實作上呢，會把資料隨機分一組一組的 batch，然後一次用一組的資料去更新 (update) 參數

    把所有的 batch 都看過一次叫做 1 epoch。

    更多的變形

    Rectified Linear Unit (ReLU)

    \[ y = c \times max(0, b + w \times x_1) \]

    可以拿這代替 hard sigmoid，變成

    \[ y = b + \sum_{2i} c_i \times max(0, b_i + \sum_j w_{ij} \times x_j) \]

    這跟上面類似的式子的統稱 Activation Function。

    Neuron (神經元)

    一個 Neuron，它執行的完整數學式子就是 (影片中的圖為例)：

    \[ a_1 = \text{sigmoid}(b_1 + \sum_j w_{1j} \times x_j) \]

    很多個 Neuron 擺在一起 -> 就變成一個隱藏層（Hidden Layer）。 把好多個層接在一起 -> 就變成了 Neural Network（神經網路），也就是現在俗稱的 Deep Learning 深度學習。

    Overfitting

    機器學習模型在訓練數據上表現變好，但對新數據沒有變好
